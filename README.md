## Waiting list

# CRUD Application React Tailwindcss, Node.js and MySQL

- Database name: waiting-list-db
- Bug in Table name clients Not clinets. 
### Install dependencies

```
# Server deps
npm install

# Client deps
cd client
npm install
```

### Run Server

```
npm run server
```

### Run Client

```
npm run dev
```

### Run Server & Client Simultaneously

```
npm run all
```
